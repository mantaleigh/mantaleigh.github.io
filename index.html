Featured Personal Project

Wellesley College Course Browser & Scheduler

A better way for Wellesley students to search for courses and plan a course schedule.

In Progress

Whenever it's time to register for classes I always get frustrated with the lack of features Wellesley's course browser has. So I made a new one. This GUI (written in Python) scrapes information from the official course browser and creates a more helpful interface.

Once completed, my version of the Wellesley College course browser will include an ability to search by day and time to make sure that the course will fit into a specific schedule, while implementing other additional search criteria. It will also provide a means to create (and save) a graphic schedule of chosen courses within the GUI.
